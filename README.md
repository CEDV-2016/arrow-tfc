# Arrow and the Cursed Kingdom

GNU/Linux minigame developed in C++ by means of [Ogre 1.9](http://www.ogre3d.org/) [Blender](https://www.blender.org/), [CEGUI](http://cegui.org.uk/) and [CEED](http://cegui.org.uk/wiki/CEED).
